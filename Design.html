Design

Le plus souvent dans l’onglet “Project”

Le design est la première étape dans le cycle concevoir-construire-tester dans l’ingénierie et la biologie synthétique.
On doit décrire ici la procédure que l’on a utilisée dans le design de nos “parts”. Il faut clairement expliquer les principes d’ingénierie utilisés pour designer notre projet.

Ce que cette page doit contenir

  • L’explication des principes d’ingénierie utilisés dans le design.
  • Une discussion sur les itérations de designs à travers lesquelles on est passées.
  • Un plan expérimental pour tester nos designs
