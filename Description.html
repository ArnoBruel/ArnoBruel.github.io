Description – Project Inspiration and Description

Dans l’onglet « Project »

Critère #4 pour la médaille de bronze :

C’est ici que l’on documente comment et pourquoi nous avons choisi ce projet iGEM. Il faut référencer le travail en dehors et au sein de l’iGEM qui nous a inspiré pour le projet, comment nous avons sélectionner notre objectif de projet, et pourquoi nous avons pensé que ce sujet était une application utile de la biologie de synthèse. Enfin, il faut donner une description claire et concise de ce que nous avons prévu de faire pour le projet.

Pour être éligible à ce critère, il est impératif d’ajouter une documentation claire à cette page.

Ce que cette page doit contenir :

  • Une description claire et concise de notre projet.
  • Une explication détaillée de ce pourquoi notre équipe a choisi de travailler sur ce projet en particulier.
  • Les références et les sources pour documenter notre recherche.
  • Utiliser des illustrations et autres ressources visuelles pour expliquer notre projet.

Conseils pour écrire cette page Project Description :

Nous sommes encouragés à mettre beaucoup d’information et de contenu sur le wiki, mais on est également encouragés à inclure des sommaires autant que possible.

Si nous pensons les sections dans la description de notre projet comme les sections d’une publication scientifique, il nous faut essayer d’être concis, précis et ne pas laisser place à l’ambiguïté dans nos accomplissements.

Références :

On est encouragés à enregistrer les références que l’on utilise tout au long de la recherche. Elles doivent être postées quelque part sur le wiki pour que les juges et les autres visiteurs puissent voir comment nous avons pensé notre projet et ce qui nous a inspiré.
